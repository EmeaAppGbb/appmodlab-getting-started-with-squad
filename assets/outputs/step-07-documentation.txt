# TaskFlow API — Endpoint Reference
## Base URL
http://localhost:3000
## Root
### GET /
Returns service metadata.
## Tasks
### GET /api/tasks
List all tasks.
### GET /api/tasks/:id
Get a single task by ID.
### POST /api/tasks
Create a new task. (title required, 1-200 chars)
### PUT /api/tasks/:id
Update an existing task. At least one field required.
### DELETE /api/tasks/:id
Delete a task.
