--- Files Created ---

The init command created the following directory structure:

.squad/
  config.json          - Squad configuration (version 1)
  team.md              - Team roster and project context
  routing.md           - Work routing rules (who handles what)
  ceremonies.md        - Team ceremonies and rituals
  decisions.md         - Decision log
  identity/
    now.md             - Current project identity
    wisdom.md          - Accumulated team wisdom
  agents/
    scribe/            - Session logging agent
      charter.md       - Agent charter/role definition
      history.md       - Agent activity history
    ralph/             - Quality/review agent
      charter.md
      history.md
  templates/           - Templates for team artifacts

.github/
  agents/
    squad.agent.md     - Copilot agent prompt for GitHub
  workflows/
    squad-heartbeat.yml     - Periodic heartbeat workflow
    squad-issue-assign.yml  - Auto-assign issues workflow
    squad-triage.yml        - Issue triage workflow
    sync-squad-labels.yml   - Sync squad labels workflow

.copilot/
  mcp-config.json      - MCP configuration
  skills/              - Copilot skills directory
